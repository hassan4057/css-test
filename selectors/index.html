Selectors

  • Element Selector :

    The element selector selects HTML elements based on the element name.

    p {
    text-align: center;
    color: red;
    }

    In this code P is called Element Selector.


  • Class Selector :

    The class selector selects HTML elements with a specific class attribute.To select elements with a specific class, write a period (.) character, followed by the class name.

    .center {
    text-align: center;
    color: red;
    }

    In this class .center is called Class Selector.


  • Id Selector :

    A CSS ID selector uses the ID attribute of an HTML element to select one unique element on a page. To use an ID selector in CSS, you simply write a hashtag (#) followed by the ID of the element. Then put the style properties you want to apply to the element in brackets.

    #para1 {
    text-align: center;
    color: red;
    }

    In this code #para1 is called id selector.
