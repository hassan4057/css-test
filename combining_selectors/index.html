Combining Selectors

  • Element with Class Selector

    IN this every P that has class big will be affected and all other P will remain unaffected.

    p.big {
    font-size: 30px;
    }

    In this code p.big is called Element with class Selector.


  • Child Selector

    Every P that is direct child of article will be affected and all other p will remain unaffected.

    article > p {
    color: red;
    }

    In this code article > p is called Child Selector.


  • Descendant Selector

    In this type of selector every p inside article element at any level whether its not direct child of article will be affected.

    article p {
    text-align: center;
    color: red;
    }

    In this code article p is called desecdant selector.
