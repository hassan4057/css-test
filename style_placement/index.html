Style Placement

There are three ways to style your html document or web page :

  • Inline Css :

    • Inline Css means that you are styling every element individually. This is the least used method for styling as it takes so much time and is not reusable.

    This whole paragraph is style by using inline css.

  • Internal Css :

    • Internal Css is used within the head tag of html file and is resuable and all the css rules are inside the style tag within the head tag.

    For example all the paragraphs other than the blue one are using internal css.

  • External Css :

    • External Css is the most used one and is the best approach for styling. IN this type we make individual file for styling and then attach that file in the head section of html page.

    The main heading of the page is style using external css.
